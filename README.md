<h2>このフォントについて</h2>
太いゴシック体をテーマに製作したフォントです。 文字の枠を横長に設定しておりどっしりとした安定感あります。
文字数も豊富に収録しているので様々な場面でお使いいただけます

<h2>収録文字</h2>
・半角英数、約物、（Google Latin Core
・ひらがな カタカナ 全角英数 全角記号 縦書き用文字
・漢字 jis第三水準まで
